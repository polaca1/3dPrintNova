# 3dPrintNova - Landing & E-commerce Platform 🖨️

## 📋 Descripción

**3dPrintNova** es una plataforma web moderna de landing page y e-commerce para una empresa de servicios de impresión 3D. Diseñada como solución completa con enfoque en experiencia visual premium, interactividad avanzada y conversión de ventas.

## ✨ Características Principales

### 🎨 Diseño & UX
- Interfaz moderna con aesthetic AI/SaaS y toques cyberpunk elegante
- Diseño responsive mobile-first optimizado para todos los dispositivos
- Tema claro/oscuro dinámico
- Animaciones avanzadas con Framer Motion y GSAP
- Scroll progress indicators y efectos parallax

### 🔧 Funcionalidades
- **Catálogo Interactivo**: Browse de productos con filtros animados
- **Builder Personalizado**: Herramienta para presupuestos en tiempo real
  - Carga de archivos STL/referencias
  - Selección de tamaño, material y color
  - Cálculo dinámico de precios
  - Preview de proyectos personalizados
- **Escena 3D Interactiva**: Visualización de impresoras 3D y productos usando Three.js
- **Social Proof**: Reviews, contadores animados, integración TikTok
- **Contacto Directo**: Integración WhatsApp con mensajes contextuales
- **SEO Optimizado**: JSON-LD para LocalBusiness y ofertas

### 🚀 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Lenguaje**: TypeScript
- **Estilos**: TailwindCSS v3 + CSS personalizado
- **Animaciones**: Framer Motion, GSAP, Lenis (smooth scroll)
- **3D**: React Three Fiber + Three.js
- **UI**: Componentes tipo shadcn + Lucide React
- **Testing**: Playwright para QA visual
- **Despliegue**: GitHub Pages (Static Export)

## 💻 Instalación & Desarrollo

### Requisitos previos
- Node.js 18+ 
- npm o yarn

### Setup
```bash
# Clonar repositorio
git clone https://github.com/polaca1/3dPrintNova.git
cd 3dPrintNova

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Visitar
# http://localhost:3000
```

### Scripts disponibles
```bash
npm run dev              # Servidor de desarrollo
npm run dev:turbo       # Desarrollo con Turbopack (experimental)
npm run build           # Build de producción
npm start               # Servidor de producción
npm run typecheck       # Verificación de tipos TypeScript
npm run lint            # ESLint
npm run visual:qa       # QA visual con Playwright
```
